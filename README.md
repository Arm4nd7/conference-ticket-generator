# Generador de Tickets

![👀](./BuscadorPaises.png)

En este proyecto pudes generar tikets con tu información mediante un formulario el cual almacenará esta información y mostrara un ticket el cual te indica el numero de ticket, nombre, foto, fecha.

###  [👀 Miralo aquí](https://arm4nd7.github.io/conference-ticket-generator/)

##  Partida 🚀
En este proyecto puedes encontar:

    ▫️ Formulario con input para datos importantes.
    ▫️ Agregacion de imagen. 
        - Redimensiona la imagen que coloques.
    ▫️Validacion de correcto ingreso de email. 
    ▫️Fondos y hoves en cada input.

## Construido con 🛠️

✔️ `HTML 5` para dar estructura a la página.<br>
✔️ `CSS` para dar estilo a la página y los componentes.<br>
✔️ `JS Vanilla` para dar dinamismo e interactividad a la página.<br>
✔️ `Diseño` uso de coleres y estilos proporcionados por [Frontend Mentor](https://www.frontendmentor.io).

## Como funciona❓
La página tiene 2 partes
1. Formulario: Informacion, imagenes, validacion de correo.
2. Generacion de Tikets: muestra de informacion y demas.

Nos centraremos en los putnos 1 y 2.
### 1️⃣
En este apartado encontraras el Formulario: 
1. **Informacion** tiene hoves para resaltar lo escrito, ademas de validacion de correo.
2. **Imagen** se puede ingresar imagenes y sera redimensionado a el espacio para el ticket.
### 2️⃣
En este apartado escontraras el tiket:
* Formacion de ticket en espacio reducido.
* Muestra de  informacion llenada en el formulario junto con la imagen ingresada.
